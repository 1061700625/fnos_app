# OESP LED 控制

OESP LED 控制是一个用于飞牛 fnOS 系统的应用，用于控制 OESP 设备上的 LED 灯。

## 作者

- 作者：小锋学长生活大爆炸
- GitHub：https://github.com/1061700625/fnos_app

## 功能特性

### 核心功能
- ✅ 控制所有 LED 的开关
- ✅ 单个 LED 精确控制
- ✅ 支持临时控制和永久设置
- ✅ 美观的 Web 界面
- ✅ 在飞牛桌面中内嵌打开（不跳转浏览器）

### 高级功能
- ✅ LED 按类型分组显示（电源指示灯、网络状态灯、硬盘活动灯等）
- ✅ 支持反向逻辑 LED（部分 LED 写入 0 点亮，写入 1 熄灭）
- ✅ 睡眠模式：在设定时间段内自动关闭所有 LED，结束后恢复状态
- ✅ 恢复默认设置：一键恢复系统默认状态
- ✅ 悬浮窗提示：优雅的用户操作反馈

### 技术优化
- ✅ 睡眠模式精确触发：替代轮询，在精确时间点触发，节省系统资源
- ✅ 优化的页面更新：只更新变化的部分，避免页面闪烁

## 安装说明

1. 在飞牛 fnOS 系统上打开应用中心
2. 点击「手动安装」
3. 上传 `oesp-led-control.fpk` 文件
4. 按照向导完成安装

## 使用说明

### LED 控制
1. 安装完成后，点击应用图标打开 Web 界面
2. 在「LED列表」标签页可以看到所有可用的 LED，按类型分组显示
3. 点击开关按钮控制 LED
4. 点击「全部打开」或「全部关闭」批量控制
5. 点击「保存配置」按钮保存当前状态，下次重启后生效
6. 点击「恢复默认」一键恢复系统默认状态

### 睡眠模式
1. 切换到「睡眠模式」标签页
2. 开启睡眠模式
3. 设置开始和结束时间
4. 在设定时间段内，所有 LED 会自动关闭，睡眠时间结束后自动恢复之前的状态
5. 时间修改后会自动保存

## 开发说明

### 项目结构

```
oesp-led-control/
├── app/
│   ├── server/     # 后端服务（Node.js + Express）
│   ├── ui/         # 应用入口配置
│   └── www/        # 前端界面（Vue.js 3）

├── config/         # 配置文件

```

### 技术栈

- 后端：Node.js + Express
- 前端：Vue.js 3
- 系统：飞牛 fnOS

### 睡眠模式实现

睡眠模式使用精确时间触发机制，而不是轮询：
1. 计算下一个开始或结束事件的精确时间
2. 使用 setTimeout 在精确时间点触发
3. 处理跨天的情况（如 22:00 到次日 06:00）
4. 配置变更时立即响应
